# VR Protocol Analyzer 使用指南

## 概述
VR Protocol Analyzer 是一个用于分析Unity VR教室场景的工具，可以扫描场景物体、分析数据并导出为JSON格式。本工具专为5小时紧急开发任务设计，由四人团队协作完成。

## 版本信息
- **工具名称**: VR Protocol Analyzer
- **版本**: 1.0
- **开发团队**: 
  - A同学（数据专家）：数据提取和JSON生成
  - B同学（界面专家）：UI界面设计和用户交互
  - D同学（性能专家）：性能监控和优化
  - C同学（集成官）：项目集成和测试
- **最后更新**: 2024-01-25

## 快速开始

### 1. 打开工具
1. 启动Unity编辑器
2. 打开目标VR教室场景
3. 在菜单栏选择: **Window → VR Protocol Analyzer**
4. 工具窗口将显示在编辑器界面中

### 2. 基本使用流程
1. **扫描场景**：点击"🔍 扫描当前场景"按钮
2. **查看物体**：在物体列表区域查看场景中的所有物体
3. **筛选数据**：使用类型筛选和搜索功能查找特定物体
4. **导出JSON**：配置导出设置，点击"🚀 开始导出JSON数据"
5. **查看结果**：在输出目录查看生成的JSON文件

## 功能模块详解

### 1. 场景扫描模块
- **功能描述**: 扫描当前激活的场景，统计物体信息
- **操作按钮**: "🔍 扫描当前场景"
- **输出信息**:
  - 场景名称
  - 物体总数
  - 各类物体统计（桌子、椅子、墙壁、灯光等）

### 2. 物体分析模块
#### 物体列表
- 显示场景中所有GameObject的详细信息
- 列信息包括：物体名称、类型、位置坐标
- 支持按名称排序

#### 筛选功能
- **类型筛选**: 按物体类型快速筛选（全部/桌子/椅子/墙壁/灯光/其他）
- **名称搜索**: 输入关键词搜索特定物体
- **实时统计**: 显示筛选后的物体数量

#### 物体操作
- **定位功能**: 点击"📍"按钮，在Scene视图中聚焦选中物体
- **类型图标**: 不同类型物体显示不同图标，便于识别

### 3. 数据导出模块
#### 导出设置
- **文件路径**: `Assets/GeneratedOutput/JSON_Exports/`
- **文件名**: 可自定义，默认`classroom_data.json`
- **数据选项**:
  - ☑ 位置信息 (Position)
  - ☑ 旋转信息 (Rotation)
  - ☑ 缩放信息 (Scale)
- **格式选项**:
  - ☑ 美化格式 (Pretty Print)
  - 缩进: 2个空格
  - 编码: UTF-8

#### 导出过程
1. 点击"🚀 开始导出JSON数据"按钮
2. 显示实时进度条，包含：
   - 当前进度百分比
   - 预计剩余时间
   - 取消按钮
3. 导出完成后显示成功对话框
4. 可选择打开输出文件夹

### 4. 性能监控模块
- **当前状态**: 由D同学开发中
- **计划功能**:
  - 扫描耗时统计
  - 导出耗时统计
  - 内存使用监控
  - 性能优化建议
  - 性能报告生成
